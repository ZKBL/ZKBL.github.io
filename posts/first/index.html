为什么要搭建博客

原因主要有两点

  • 最近迷上了用文字记录一些东西
  • 发现自己工作时很多东西总是查了又查，却一直没有记录和总结，所以想写博客来记录跟总结自己学到的东西

技术选型

作为一个饿着肚子点外卖都能选择半小时的深度选择困难症患者，选择博客的搭建方式纠结了很久。动态站点类型之前用过 WordPress 感觉有点臃肿，太多东西自己用不到。又因为自己使用 Obisidian 记笔记，所以优先考虑使用原生支持 Markdown 的框架，最后在 hexo 和 hugo 之间纠结，查了不少资料，也参考了不少人的意见。终于决定使用 hugo 加上 loveit主题
